# 企业级类扣子、Dify AI 应用引擎架构设计与实践面试专项突击

## 1. 项目概述
打造一个与扣子(Coze)、Dify 类似、功能齐全的企业级 AI 应用引擎平台。集成可视化工作流编排、大模型调用、RAG 知识库检索等核心功能，支持将编排好的 AI 智能体发布为 API 服务。

技术栈：Next.js 16 + React 19 + NestJS + PostgreSQL + Qdrant + LangChain.js + LangGraph.js

## 2. 核心业务需求
- 可视化工作流编排：拖拽式 DAG 流程设计
- 多模型集成：Ollama 本地/云端 LLM 调用，Prompt 编排
- RAG 知识库管理：文档解析 → 文本分块 → 向量化 → 检索
- API 服务化 (BaaS)：发布为 API，支持 SSE 流式响应
- 全链路监控：实时执行日志、Token 消耗统计、节点级耗时分析

## 3. 技术架构
### AI 引擎核心 SDK
- 自研引擎：基于 DAG 和 Kahn 算法的 TypeScript 工作流执行引擎
- LangChain/LangGraph：高级 Agent 模式（循环、中断、状态管理）
- 可视化编辑器：XYFlow (React Flow) + Tiptap

### 前端应用
- Next.js 16 (App Router / RSC)
- React 19 + XYFlow + Tiptap
- Tailwind CSS & Shadcn/ui
- Prisma ORM

### 后端服务
- NestJS + RxJS
- Passport & JWT 认证
- BullMQ (Redis) 异步任务

### 基础设施
- Ollama / PostgreSQL / Qdrant
- Docker & Docker Compose
- TurboRepo Monorepo
- Husky & Commitlint

## 4. 简历描述模板

### 项目描述 STAR 方法论
- **Situation（背景）**：为什么要做这个项目
- **Action（我的职责）**：我具体做了什么
- **Result（成果）**：取得了什么可量化的成果

### 初级/中级描述
- 基于 XYFlow 实现工作流编辑器核心交互
- 集成 Tiptap Prompt 变量编辑器，支持 ${node.var} 格式
- 使用 Tailwind CSS 和 Shadcn/ui 构建 UI
- 负责知识库模块前端开发

### 中级/高级描述
- 自主设计并实现了基于 DAG 的工作流执行引擎，利用 Kahn 算法实现拓扑排序与循环检测
- 设计了基于注册模式的节点扩展系统
- 基于 LangChain 和 Qdrant 搭建 RAG 管道
- 在 NestJS 中实现 SSE 流式响应
- 采用 Monorepo 架构 + Docker 容器化部署
